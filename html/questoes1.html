Questões de Programação Orientada a Objetos (POO) - Múltipla Escolha

1. O que é Programação Orientada a Objetos?

  • a) Um paradigma de programação baseado em objetos
  • b) Um método de programação baseado em funções
  • c) Um sistema de codificação orientado a objetos
  • d) Um sistema de codificação baseado em procedimentos

Resposta Correta: a) Um paradigma de programação baseado em objetos

2. Quais são os principais conceitos da POO?

  • a) Encapsulamento, Herança e Polimorfismo
  • b) Iteração, Condicionamento e Abstração
  • c) Modularidade, Abstração e Encapsulamento
  • d) Modularidade, Herança e Polimorfismo

Resposta Correta: a) Encapsulamento, Herança e Polimorfismo

3. Qual é a diferença entre uma classe e um objeto?

  • a) Uma classe é uma instância de um objeto
  • b) Uma classe é um modelo para criar objetos
  • c) Um objeto é um modelo para criar classes
  • d) Um objeto é um método dentro de uma classe

Resposta Correta: b) Uma classe é um modelo para criar objetos
