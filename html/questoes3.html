Questões de Desenvolvimento Web - Múltipla Escolha

1. O que significa HTML?

  • a) Hyper Text Markup Language
  • b) High Tech Machine Learning
  • c) Home Tool Management Language
  • d) Hyperlink and Text Management Language

Resposta Correta: a) Hyper Text Markup Language

2. O que é CSS?

  • a) Computer Style Sheets
  • b) Creative Style Selector
  • c) Cascading Style Sheets
  • d) Colorful Style Sheets

Resposta Correta: c) Cascading Style Sheets

3. O que é JavaScript?

  • a) Uma linguagem de programação de servidor
  • b) Uma linguagem de programação de banco de dados
  • c) Uma linguagem de programação de front-end
  • d) Uma linguagem de programação de estilos

Resposta Correta: c) Uma linguagem de programação de front-end
