Questões de Banco de Dados - Múltipla Escolha

1. O que é um banco de dados?

  • a) Uma aplicação web
  • b) Um sistema de arquivos
  • c) Um conjunto organizado de dados
  • d) Um tipo de linguagem de programação

Resposta Correta: c) Um conjunto organizado de dados

2. O que é um SGBD?

  • a) Um sistema gerenciador de banco de dados
  • b) Um sistema gráfico de banco de dados
  • c) Um sistema de busca de dados
  • d) Um tipo de linguagem de programação

Resposta Correta: a) Um sistema gerenciador de banco de dados

3. O que é SQL?

  • a) Structured Language Query
  • b) Structured Query Language
  • c) Simple Query Language
  • d) Scripted Query Language

Resposta Correta: b) Structured Query Language
